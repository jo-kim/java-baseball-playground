+ 숫자 야구 게임 기능 구현 목록
    + 1~9 사이 숫자 3개를 랜덤으로 뽑는다.
    + 3자리 수를 입력 받는다.
        + 받은 값이 3자리가 아닌 경우 에러
        + 받은 값이 숫자가 아닌 경우 에러
        + 받은 값 중 중복값이 있는 경우 에러
    + 입력 값에 따라 힌트 구현
        + 자리와 숫자값이 맞을 경우 - strike
        + 숫자만 맞을 경우 - ball
        + 둘다 틀릴경우 - nothing
    + 게임의 승패를 구분
        + 3 스트라이크 게임 종료
